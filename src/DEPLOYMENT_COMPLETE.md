### Build Error on Vercel


### Supabase Connection Error
**Symptom:** "Failed to fetch" or database errors
**Solution:** Check `/utils/supabase/info.tsx` has correct credentials

### Admin Portal Not Working
**Symptom:** Can't access `/admin`
**Solution:**

### Payment Gateway Fails
